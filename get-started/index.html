# 快速上手

# npm 安装

npm i smart-vue-ui -S

# 本地引用

官方仓库下载或使用命令行克隆到本地

$ git clone github.com:qiujia236/wheel-demo-1.git

# 开始使用

完整版引入：

import Vue from 'vue'
import smartUI from 'smart-vue-ui'
import 'smart-vue-ui/dist/smart-vue-ui.css'

Vue.use(smartUI)
