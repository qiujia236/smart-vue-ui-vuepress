# icon

提供一些常用图标集合。

# 基本用法

示例代码：

<my-icon name="left" color="red" />
<my-icon name="right" color="#1abc9c" />
<my-icon name="up" color="#2ecc71" />
<my-icon name="down" color="#3498db" />
<my-icon name="download" color="#9b59b6" />
<my-icon name="setting" color="#f1c40f" />
<my-icon name="thumsUp" color="#e67e22" />
<my-icon name="loading" color="#e74c3c" />
<my-icon name="error" color="#95a5a6" />
<my-icon name="success" color="#f39c12" />
<my-icon name="search" color="#c0392b" />
<my-icon name="circle" color="#bdc3c7" />

# Attributes

组件将被渲染为button标签，因此你可以使用所有的button原生属性。

参数 说明 类型 可选值 默认值
name icon 图标名 String left，right，up，down，download，setting，thumsUp，loading，error，success，circle --
color 填充颜色 String -- --
